# Yii2 Demo

## 🚀 Запуск проекта

Для запуска используйте Docker:

docker-compose up

Если у вас Docker Compose V2:

docker compose up

---

## 🗄 Наполнение базы данных (Seeder)

Для заполнения базы тестовыми данными выполните:

docker exec -it yii2_app php yii seed/run

---

## 🧪 Запуск тестов

Для запуска функциональных тестов:

vendor/bin/codecept run functional controllers/AlbumControllerCest

Если модуль REST ещё не установлен:

composer require codeception/module-rest --dev

---
